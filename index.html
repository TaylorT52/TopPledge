Google Sheets Setup

To make this leaderboard live, use either Name + Points or First Name + Last Name + Points. An optional Note column still works too.

  1. Put your pledge names and totals into a Google Sheet.
  2. Make the sheet public or use File > Share > Publish to web so the browser can read it.
  3. Open this file and paste either your normal Google Sheets link into CONFIG.googleSheet.sheetUrl or a published CSV link into CONFIG.googleSheet.publishedCsvUrl.
  4. Host the page with GitHub Pages, Netlify, or any simple web server so the browser can fetch the sheet.

If you would rather show dollars later, switch scoreFormat back to "currency" and update scoreLabel.
